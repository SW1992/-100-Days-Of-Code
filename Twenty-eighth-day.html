CSS Pre-processors & LESS

CSS pre-processors are style sheet languages that can be used to extend plain CSS allowing a developer to write cleaner code & save time, by storing & re-using functionality.

LESS

One of the more popular pre-processors, is LESS, often compared to the also popular Sass.

LESS, was first built with the Ruby programming language like Sass, but is now based in Javascript.

Both can be thought of as scripting languages that aim to make CSS more powerful, versatile & customizable.

Using LESS

To use LESS you must first save your CSS file with the ".less" extension.

It's possible to use it locally, on the client-side, by linking to it in your page, like you would a normal CSS stylesheet, changing the rel attribute slighty.

<link rel="stylesheet/less" type="text/css" href="styles.less"/>

You also need to download the less.js file from the http://lesscss.org/ site, and follow your stylesheet/stylesheets with a script tag like this.

<script src="less.js" type="text/javascript"> </script>

For production, if your new to pre-processors like me, it's recommended you use 3rd party compiliation software to install it. Such as Koala or Codekit, or one of the many others.
