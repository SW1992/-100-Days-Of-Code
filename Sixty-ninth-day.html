Chrome Dev Tools: Application

The Application tab, formerly the resources tab, can be used to inspect what external resources the current page is loading.

The frames pane can be used to view resources, organized into frames. With categories like top, which is the main document, sub frames of the main document and their resources, and below it other resources of the main document like images and scripts.

Other resources can also be viewed in the application pane, things like external SQL data, as well as local storage data.
