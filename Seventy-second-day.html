Chrome Dev Tools: Console API Commands

There are many different API commands you can use in the Chrome Dev Tools console, to do various things.

console.assert(some expression, message) can be used to ouput an error message to the console, when the expression given to it evaluates to false.

console.clear() can be used to clear the consoles log history, providing preserve log isn't enabled.

console.dir(object) will print a JS representation of a given object.

console.count(label, string) can be used to log a statement to the console, and keep track of the amount of times the methods been invoked with the same string input

console.group(title) & console.groupEnd() can be used to visually tie together a group of logged statements, under a given title, in the console.

These are just some of the popular Console API commands.
