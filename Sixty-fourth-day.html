Chrome Dev Tools: Elements

You can use the elements tab in the chrome developer tools to tweak the CSS rules & HTML mark-up of your webpage/website.

As well as basic things like changing the background colour property of specific elements you can also tweak their layout.

You can do this through a box model in the computed pane, that lets you change values for properties like, border, margin & padding.

Edits are viewable in real time, so you can see the effect changes would have on your webpage, as you change property values.
