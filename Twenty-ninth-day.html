LESS Variables & Mixins

LESS Variables

Variables in LESS, are much like variables in other programming languages. they can be used to store some property value, for instance a colour or font-size, so it can be re-used on different elements in the document.

You can use them to save values for setting CSS properties such as font-family, height & width etc.

LESS Variables save a developer the time of writing the same property value for different selectors in CSS & can help your code follow the DRY principle.

LESS Mixins

Where as variables are useful for letting you store & reuse a single property value in CSS, mixins allow you to store & re-use an entire set of properties from one rule-set to the next.

So, Mixins can be used for setting the font-family, styling borders etc, across rule-sets.
