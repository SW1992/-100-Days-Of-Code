Chrome Dev Tools: Network

The network tab grants access to a detailed log of information about the time it takes resources in your webpage or app to load.

If your app or page was suffering with sluggish performance the network tab can be useful to diagnose the root cause of the problem.

It offers a handy overview graph that can be used to deduce the resource, or group of resources that are to blame, with pin point accuracy.

You can click an item in the requests table to get more detailed breakdown of information about a particular resource.

General information like what HTTP method was used in the request, the status code of the response, a further breakdown of timing information etc.

You can can also shift click and hover over items in the request table to determine the initiator of a request, resource dependencies etc.
