BEM

BEM & it's uses.

BEM or Block, Element, Modifier is a CSS naming convention which defines a set of rules for developers to use when naming their CSS classes.

It aims to make codebases more maintainable & easier to read, paticulary on large projects, defining a rigid system for developers & development teams to follow.

And from personal expierence, when it comes to programming naming things can be hard, BEM, aswell as other methodologies like OOCSS (Object Oriented CSS) aim to remedy this problem.
