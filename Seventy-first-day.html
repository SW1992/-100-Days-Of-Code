Chrome Dev Tools: Console

The chrome dev tools console is a useful tool for debugging JavaScript, you can use console specific commands like console.log() or console.error() to help you do this.

You can view diagnostics of the page you are running, you can look at structured data you have got back from API calls, measure execution time, view tracked exceptions etc.

To temporarily save a history of information outputted in the console, you can click preserve log, it will then be persisted even if you refresh or change the page, you can click save as to permanently save a log.
